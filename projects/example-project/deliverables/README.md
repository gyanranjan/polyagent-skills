## Deliverable Types

| Type | Description | Typical Producing Role |
|------|-------------|----------------------|
| Opportunity Brief | Market and user opportunity framing | entrepreneur |
| Product Brief / PRD | Product requirements and scope | product-manager |
| Architecture Design | System component and integration design | systems-architect |
| Implementation Plan | Engineering task breakdown | engineering-team |
| Test Plan | QA test strategy and cases | qa-validator |
| Security Review | Threat model and findings | security-guardian |
| Operations Design | Deployment and monitoring plan | operations-commander |
| Growth Model | Funnel and experiment design | growth-engineer |
| Research Report | Evidence gathering and synthesis | research-analyst |
| Decision Record | Committed decision with rationale | any role |

## Naming Convention

Files in this directory should follow:
```
[YYYY-MM-DD]-[type]-[brief-name].md
```

Examples:
- `2026-03-06-opportunity-brief-restaurant-supply-chain.md`
- `2026-03-06-architecture-design-mvp-v1.md`
- `2026-03-06-security-review-api-auth.md`
