# Agent Instructions — polyagent-skills

You have access to a portable skill library in `skills/`. Shared conventions are in `common-skills/`.

For any task, first check `skills/` for a matching skill. Read its SKILL.md and follow the instructions. Apply common-skills when referenced.

Available skills: requirement-study, implementation-sketch, mail-summarizer, document-analyzer, deck-creator, repo-bootstrap, agent-writer, desensitizer, remote-ops.

Each skill directory contains a SKILL.md with: Purpose, When to Use, Inputs, Process steps, Output Format.
