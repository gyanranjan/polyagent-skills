# Open Questions: Example Project

_Log questions that need answers before work can proceed. Any role can add questions. Assign them to the role best positioned to answer. Remove questions when resolved — record the answer in the relevant role's files or in final-decisions.md._

---

## Question Log

### Template for adding a question:
```
### Q-NNN: [Question title]
- **Question**: [Precise question]
- **Asked by**: [Role]
- **Needed for**: [What depends on the answer]
- **Assigned to**: [Role best positioned to answer]
- **Priority**: BLOCKING / HIGH / MEDIUM / LOW
- **Status**: OPEN / IN PROGRESS / RESOLVED
- **Resolution**: [When resolved, add answer here with date]
```
