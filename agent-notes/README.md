# Agent Notes

Cross-cutting observations, quirks, and tips about specific AI coding agents.

These notes capture practical knowledge from using different agents with the polyagent-skills library. Unlike skills (which are instructions FOR agents), these are notes ABOUT agents — for human reference.

## Structure

One file per agent or per topic:

- `claude-code-notes.md` — Claude Code quirks and tips
- `kiro-notes.md` — Kiro quirks and tips
- `codex-notes.md` — Codex quirks and tips
- `cross-agent-comparison.md` — How agents differ on common tasks

## Contributing

When you discover an agent-specific behavior that affects how skills work, add it here. If it's a bug or limitation, also add it to `KNOWN_ISSUES.md`.
